# 🍛 Curry Point - Food Ordering Website

## 📌 Project Overview
Curry Point is a simple restaurant website that allows users to explore food items, view menus, and experience an interactive food ordering interface.  
The project focuses on creating a responsive and user-friendly UI for an online food ordering system.

This project demonstrates front-end development concepts using HTML, CSS, and JavaScript.
